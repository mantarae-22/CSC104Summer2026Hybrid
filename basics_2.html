CSC104 HTML Basics

HTML Info

HTML stands for HyperText Markup Language. HTML provides the structure for our webpages. It can contain text, images, video within the tags. There are numerous amounts of tags that you can use. HTML does not have many rules, but has many best practices.

HTML Versions

  • HTML 1.0 - developed in the early 1990's
  • HTML 2.0 - released in 1995
  • HTML 3.2 - released in 1997
  • HTML 4.01 - released in 1999
  • HTML 5 - released in 2014 (this is the current version)

CSS Info

CSS stands for Cascading Style Sheets and makes our HTML content look better. CSS can control layout, colors, font - any of our layout and design principles. Any default browser styling can be changed with CSS.

CSS Versions

  • CSS1 - released in 1996
  • CSS2 - released in 1998
  • CSS3 - released in 2011 (this is the current version)

Basic Structure

        selector{
            property: value;
        }
  • Selector - can be any HTML element, Custom class or an ID of an HTML element
  • Property - design element we want to change
  • Value - whatever we want to change the design element to

Ways to Include CSS in our Webpage

  • External - this is the best practice - for example, main.css
  • Embedded - styles are located in the webpage
  • Inline - add CSS style(s) to a particular HTML element
